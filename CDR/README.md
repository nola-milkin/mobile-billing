### Description
Программа для тарификации абонентов с заданным тарифом при помощи файла с CDR в формате .csv. 

### Run
Для запуска необходим Python любой версии, используются только стандартные его библиотеки. В случае запуска под ОС Linux достаточно просто запустить файл (присутствует шибенг).
```
python3 ./cdr.py
```
### Input files
По умолчанию программа будет искать файл с CDR как `data.scv`. Другое название или путь можно передать как аргумент в программу или задать в переменной `CDR_FILE`, объявленной в начале файла. 
```
python3 ./cdr.py another.scv 
```
