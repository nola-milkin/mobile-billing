### Description
Программа для тарификации абонентов с помощью файла со статистикой IP-трафика в формате NetFlow.

### Run
Для запуска программы необходим:
- Python3 с библиотекой `matplotlib`
```
pip3 install matplotlib
```
- утилита nfdump
```
apt install nfdump # аналогично для pacman и т.д.
```
В случае запуска под ОС Linux достаточно просто запустить файл (присутствует шибенг).
```
python3 ./netflow.py
```

### Input/Output files
Файл с трафиком NetFlow не загружен в репозиторий в связи с наличием конфиденциальной информации внутри, его необходимо добавить.

По умолчанию программа будет искать файл с NetFlow как `nfcapd.202002251200`. Другое название или путь можно передать как аргумент в программу или задать в переменной `NETFLOW_FILE`, объявленной в начале файла.
```
python3 ./netflow.py nfcapd.123 
```
Затем с помощью утилиты `nfdump` (использовалась версия `1.6.18-2`) создаётся дамп в текстовом виде и сохраняется в файл `dump.txt`. В последующем программа будет использовать его для работы.
